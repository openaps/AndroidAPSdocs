# Syai X1


## 使用 X1 與 Syai Tag 應用程式

-   從 <https://play.google.com/store/apps/details?id=com.syai.tag> 安裝 apk。

-   Start X1 傳感器

- 在 [組態建置工具，BG 資料來源](#Config-Builder-bg-source) 中選擇 Syai Tag。

在 Syai 應用程式中啟用廣播：

1. 選擇設定檔
2. Syai 健康生態系統
3. 輕觸立即連線到 AAPS，接受第三方資料轉移協議
4. 啟用與 AAPS 共享資料
