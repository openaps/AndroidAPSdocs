# Ottai(歐態) M8


## 使用 M8 配合 歐態（Ottai）應用程式

-   從 <https://play.google.com/store/apps/details?id=com.ottai.seas> 下載並安裝 apk。

-   啟動傳感器

- 在 [組態建置工具，BG 資料來源](#Config-Builder-bg-source) 中選擇 Ottai。

在 Ottai 應用程式中啟用廣播：

1. 選擇設定檔
2. Ottai 集線器
3. 輕觸與 AAPS 共享，接受第三方資料轉移協議
4. 啟用與 AAPS 共享血糖資料
