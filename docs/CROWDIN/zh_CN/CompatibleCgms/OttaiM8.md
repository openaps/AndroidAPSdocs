# Ottai M8


## 搭配Ottai应用程序使用M8

-   从<https://play.google.com/store/apps/details?id=com.ottai.seas>下载并安装该apk文件。

-   启动传感器

- 在[ConfigBuilder，BG源](#Config-Builder-bg-source)中选择Ottai。

请在Ottai应用中启用广播功能：

1. 选择 Profile
2. Ottai hub
3. 点击"与AAPS共享"，同意第三方数据传输协议
4. 启用"与AAPS共享血糖数据"功能
